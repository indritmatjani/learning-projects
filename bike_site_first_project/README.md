# AOD New York - Vintage Bike Shop

A responsive landing page for a vintage bicycle shop located in New York, showcasing custom bike builds and modifications.

## 🔗 Live Demo

[View Live Site](https://indritmatjani.github.io/bike_site_first_project/)

## 📝 About

This project is a static landing page created during a web design course to practice fundamental HTML and CSS layout techniques. The site presents AOD New York, a fictional vintage bicycle shop, displaying their custom bike builds, modifications, and product gallery.

The main focus of this project was mastering precise image positioning and creating complex layouts using traditional CSS techniques.

## 🛠️ Technologies Used

- HTML5
- CSS3

## 💡 Key Learning Outcomes

Through building this project, I developed proficiency in:

- **CSS Float Layouts**: Using `float: left` and `float: right` to create multi-column layouts and position elements side by side
- **Background Image Techniques**: Implementing `background-image` with `background-size: cover` and `background-position` for responsive image displays
- **CSS Positioning**: Applying `position: relative` to precisely place elements and icons throughout the page
- **Layout Structure**: Creating complex page sections using div-based layouts and managing element spacing

## 🚀 How to View

### Option 1: View Online
Simply click the [live demo link](https://indritmatjani.github.io/bike_site_first_project/) above to see the site in your browser.

### Option 2: Run Locally
1. Clone this repository or download the ZIP file
   ```bash
   git clone https://github.com/indritmatjani/bike_site_first_project.git
   ```
2. Navigate to the project folder
3. Open `index.html` in your web browser (double-click the file or right-click → Open With → Your Browser)
